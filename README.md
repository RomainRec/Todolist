## Etapes d'utilisation :

Pour pouvoir profiter des gems, n'oublie pas un petit :
```
   $> bundle install
```

Ensuite : 
```
   $> rails db:create
```

Et: 
```
   $> rails db:seed
```

Puis : 
```
   $> rails db:migrate
```

Et enfin  : 
```
   $> rails s
```
